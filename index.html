• git clone путь к репозиторию
  • git status - статус текущего локального репозитория
  • git add файл(если в другой папке, то указать путь) - добавить новый файл под контроль гита
  • git commit -ad "Комментарий" - зафиксировать изменения и дать комментарий
  • git push - отправть все закоммиченные изменения в репозиторий на сервер
  • git pull - забрать все свежие обновления
